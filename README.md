# bot-src
Main source code of discord bot (done in python)

using python 3.13.1

## Env Variables

```txt
DISCORD_BOT_TOKEN={bot token}
REDIS_URL={connection string to redis}
DISCORD_GUILD_ID={server id}
GOOGLE_API_KEY={gemini api key}
OPENAI_API_KEY={openai api key}
```

## Command

```bash
python main.py
```
